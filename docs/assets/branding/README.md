# VIPP logo

VIPP stands for **Visual Image Processing Platform**. Its tagline is **Visual
workflows for reproducible bioimage analysis.**

The VIPP mark is a **voxel V**: connected square nodes form the initial of the
platform while also suggesting image pixels moving through a visual workflow.
The two branches converge on one bright output voxel, which represents a
processed result and a point of focus.

## Files

- `vipp-logo.svg` — primary horizontal logo for light backgrounds.
- `vipp-logo-dark.svg` — horizontal logo on a solid dark field.
- `vipp-mark.svg` — square app, plugin, profile, and favicon symbol.
- `vipp-logo-mono.svg` — one-color version for print, embossing, and restricted palettes.

All lettering is outlined as SVG paths. There are no embedded images, external
fonts, filters, or proprietary features. Named groups separate the symbol and
wordmark, and the geometry uses ordinary paths, rectangles, and circles so it
can be edited in Inkscape, Illustrator, Figma, or a text editor.

## Practical use

Keep clear space around the logo equal to roughly one node width. Prefer the
full horizontal logo above 180 px wide and the standalone mark below that. The
mark remains legible down to about 24 px; at 16 px, remove the dark inner dot
from the output node if needed.
